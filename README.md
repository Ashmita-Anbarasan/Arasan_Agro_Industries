HOW TO VIEW THE WEBSITE:
------------------------

1. Download EVERYTHING from this folder:
   - arasan-rice-mill-harvest-fields.html (the main website file)
   - The entire "images" folder with all photos inside

2. Keep the files organized like this on your computer:
   
   YourFolder/
   ├── arasan-rice-mill-harvest-fields.html
   └── images/
       ├── Arasan_Logo_1_.png
       ├── photio_image__1_.jpg
       ├── photio_image__2_.jpg
       ├── (and all other images...)

3. Simply double-click "arasan-rice-mill-harvest-fields.html" 
   It will open in your default web browser with all images showing!

IMPORTANT:
----------
- The HTML file and "images" folder MUST be in the same location
- Don't rename the "images" folder
- All image files must stay inside the "images" folder
